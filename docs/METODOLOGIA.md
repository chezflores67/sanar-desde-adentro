# Metodología y límites

Las cinco heridas se usan como marco metafórico y divulgativo. No constituyen diagnósticos ni categorías clínicas validadas.

Los ejercicios integran principios generales de terapia cognitivo-conductual, terapia de esquemas, terapia de aceptación y compromiso y autocompasión. La aplicación no administra pruebas clínicas validadas ni sustituye evaluación profesional.
