# Política de privacidad — versión de validación

La versión v0.1 almacena perfil, respuestas, progreso y bitácora en el navegador del dispositivo mediante `localStorage`. La aplicación estática no envía esa información a SMART APPS ni a GitHub.

El usuario puede descargar un respaldo JSON, restaurarlo y borrar todos sus datos. Borrar datos del navegador también puede eliminar el progreso.

Antes de cualquier lanzamiento comercial se deberá realizar revisión jurídica y técnica.
